Thumb Controls

Calibration controls

This section describes the calibration control bindings hard-coded in Thumb. The number of complexity of these may at first seem ridiculous. However, these bindings are not active under normal circumstances, and are hidden behind an explicit calibration mode. They are here for experts, and never need to be touched by the vast majority of users.

Calibrating a Varrier or a multi-projector dome can be an arduous task. Years of experience have shown that a solid set of keyboard bindings make the performance of these tasks far quicker and easier than any GUI interface. While it takes longer for the beginner calibrator to learn to perform a calibration, the keyboard allows the expert calibrator to work very quickly without the need to shift attention from the display to the interface.

Calibration state controls

Control-TabToggle calibration mode. In calibration mode, the normal distribution of events is interrupted. Instead of being sent to the application, events are routed instead to the application configuration. This allows view frusta and display modes to be tuned interactively.
Control-SpaceCycle forward through the list of configurable displays.
Control-BackspaceCycle backward through the list of configurable displays.

Frustum calibration controls

The value of d depends upon the state of the Caps Lock modifier key. Caps Lock OFF gives coarse control with d = 0.5, while Caps Lock ON gives fine control with d = 0.05. While in calibration mode...

Control-LeftYaw decrease d
Control-RightYaw increase d
Control-UpPitch decrease d
Control-DownPitch increase d
Control-PageUpRoll decrease d
Control-PageDownRoll increase d
Control-InsertHorizontal FOV increase d
Control-DeleteHorizontal FOV decrease d
Control-HomeVertical FOV increase d
Control-EndVertical FOV decrease d
Control-Shift-LeftTheta increase d
Control-Shift-RightTheta decrease d
Control-Shift-UpPhi increase d
Control-Shift-DownPhi decrease d
Control-Shift-PageUpRho increase d
Control-Shift-PageDownRho decrease d

Varrier calibration controls

While in calibration mode, with a Varrier display configuration selected...

Control-LeftShift decrease 0.00005
Control-RightShift increase 0.00005
Control-UpOptical thickness decrease 0.0001
Control-DownOptical thickness increase 0.0001
Control-Shift-LeftAngle decrease 0.01
Control-Shift-RightAngle increase 0.01
Control-Shift-UpPitch increase 0.01
Control-Shift-DownPitch decrease 0.01
